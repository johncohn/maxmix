## Approved Architecture
- Vanilla JS (ES2020), single `index.html`, no framework, no build step, no npm
- Two-video swap (NOT single-video src-swap — produces 100–400 ms black-frame gap)
- Never set `loop` attribute — suppresses `ended` event that drives sequencing

## Key Anti-Patterns (Enforced)
- No MSE/SourceBuffer — plain MP4 with two-video swap only
- No npm/frameworks
- Never `loop` on video elements
- Always `muted + playsinline`
- No blind `git add .` or `git add -A`

## Sequencing Algorithm
```javascript
const CLIPS = [1, 2, 3, 4, 5];
function pickExcept(exclude) {
  const pool = CLIPS.filter(n => n !== exclude);
  return pool[Math.floor(Math.random() * pool.length)];
}
function buildSong() {
  const s1 = CLIPS[Math.floor(Math.random() * CLIPS.length)];
  const s2 = pickExcept(s1);
  const s3 = pickExcept(s2);
  const s4 = pickExcept(s3);
  return [`clips/c${s1}_s1.mp4`, `clips/c${s2}_s2.mp4`, `clips/c${s3}_s3.mp4`, `clips/c${s4}_s4.mp4`, `clips/c5_s5.mp4`];
}
```
